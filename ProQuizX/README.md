# ProQuizX - Professional Quiz Application

A fully functional quiz platform built with Flask, SQLite, HTML, CSS, and JavaScript.

## Features
- User registration and login
- Forgot password simulation
- Secure password hashing
- Session-based authentication
- Responsive modern UI
- Quiz categories
- Dynamic questions from SQLite database
- Quiz timer with auto-submit
- Next/Previous navigation
- Result page with percentage and pass/fail status
- Correct answer review
- Score history

## How to Run

```bash
cd ProQuizX
pip install -r requirements.txt
python run.py
```

Open this in browser:

```text
http://127.0.0.1:5000
```

## Default
Create a new account from Sign Up page and start quiz.
